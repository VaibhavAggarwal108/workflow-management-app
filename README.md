# Workflow Management Application

A full stack workflow management system built with Spring Boot, Angular, and PostgreSQL.

## Features
- JWT authentication
- Role based workflow actions
- Draft editing and status transitions
- Workflow history tracking
- Dashboard with summary metrics
- Search, sorting, and filtering

## Tech Stack
- Frontend: Angular
- Backend: Spring Boot
- Database: PostgreSQL

## Local Setup
### Backend
Run the Spring Boot server from the backend folder.

### Frontend
Run the Angular app from the frontend folder.

### Database
Use PostgreSQL and configure the connection in application properties or environment variables.
